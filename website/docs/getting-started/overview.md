# stimm

Stimm is an Optimistic VUI runtime on top of LiveKit Agents.

- A low-latency `VoiceAgent` handles live speech interaction.
- A high-capability `Supervisor` reasons in the background and steers behavior.
- Both exchange typed messages through the stimm protocol.

Use Stimm when you want immediate conversational response with deeper
tool-augmented reasoning in parallel.

Optimistic VUI means the system starts behaving usefully as soon as it has
enough confidence to move the interaction forward: acknowledge early, speak
early, and keep reasoning in parallel.
