# stimm

Stimm is a dual-agent voice orchestration layer on top of LiveKit Agents.

- A low-latency `VoiceAgent` handles live speech interaction.
- A high-capability `Supervisor` reasons in the background and steers behavior.
- Both exchange typed messages through the stimm protocol.

Use Stimm when you want immediate conversational response with deeper tool-augmented reasoning in parallel.
